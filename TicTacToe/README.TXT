# Tic-Tac-Toe Game

## Author
- Priyanshu Chaudhary

## Description
This is a simple graphical Tic-Tac-Toe game created in Java using the Swing library. Two players can take turns marking the 3x3 grid with "X" and "O" to compete for a win. The game checks for wins and draws, and it displays the winner in a dialog box.

## How to Play
1. Download or clone the repository to your local machine.
2. Compile and run the Java program using a Java IDE or command line.
3. Players take turns clicking on the empty cells to make their moves.
4. The game checks for a win or a draw and displays the result in a dialog box.
